# @yes-human/packs

Built-in domain workflow packs for yes-human SDK.

## Installation

```bash
npm install @yes-human/packs
```

## Available Packs

1. **developer-pack**: Code review, bug fix, explain code, test generation, security reviews.
2. **document-pack**: Summarization, action items extraction, reports, outlines, comparisons.
3. **business-pack**: Business plans, financial sheets, investor teaser slide structures.
4. **security-pack**: Prompt injection scans, dependency checks, secrets leaks.
5. **startup-pack**: PRDs, launch plans, feature roadmaps.
6. **default-pack**: Contains 10 light, multi-purpose workflows.
